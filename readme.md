# Pongu!

Pongu! game created during pebble {code} hackday!

### Includes
* Local server
* LiveReload
* Sass
* Jshint
* CSS Grid
* CSS Reset

### Installation

Install [node](http://nodejs.org/) & [gulp](http://gulpjs.com/)

`brew install node`, `npm install -g gulp`

Install packages

`npm i`

Build assets (only required once)

`gulp build`

Run Gulp

`gulp`
